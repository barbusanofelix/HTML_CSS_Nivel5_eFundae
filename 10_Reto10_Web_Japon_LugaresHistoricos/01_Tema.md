# Que considerar en este proyecto
Crear una pagina web sobre sitios con relavancia Historica en Japon: Fichas sobre enclaves con relevancia Historica en Japon.  Colocar como ejemplo los castillos de Tsuruga, Higashiyama y Matsuyama.

## Que información mínima hay que incluir:   
información sobre los lugares:  
    Titulo, una foto, horario de apertura y además,  un contenido extra, con una reseña histórica, para quien quiera ampliar la información.

Información General o que es compartida para todas las fichas:  
    Contacto y medios de transporte para acceder.

Lo que tenemos que hacer en concreto es crear la pagina que contenga : [Encabezado],  [Navegación],  [Sección principal], [Artículos],  [Barra lateral],  [Pie de página] , usando las etiquetas semánticas de HTML en cada sección ( donde aplique)

Algunas personas tienen visión reducida asi que a esas personas les importa mucho optimizar la accesibilidad.

La idea es que lo hagamos PASO a PASO. Aplicar las mejores practicas para Junior ( tal vez Junior +) .

Para el CSS NO usar Bootstrap. Las Clases con la metodología BEM.

Usar las etiquetas semánticas de HTML.

Usar grid Layout y Media Query para los dispositivos mas usuales. Incluyamos un caso de voltear la pantalla del móvil o la table.


Si crees que es muy complejo hacerlo todo de una vez pues vayamos , como ya indicado, PASO a PASO, de tal forma que yo vaya entiendo como desarollarlo y

porque hacerlo asi.

Manten integridad en el código que vayamos usando ( es decir, no cambiar los nombres o estructuras que hayamos creado en códigos que hayamos hecho y hayamos
tenido que repetirlo ).

Usar comentarios en buena cantidad para que cualquiera que vea el código  lo entienda.


De ser posible:


En el encabezado, incluir varios niveles y las etiquetas correspondientes:
En Navegacion:  
Incluir el elemento de navegación con varios enlaces a otras secciones y las etiquetas correspondientes, ( Se vez necesario hacer
mas de una pagina, pues adelante),   
Seccion principal:  
Incluir la seccion principal con la etiqueta correspondiente con varios elementos graficos y/o  listados que faciliten la lectura.

Articulos:  
Incluir varios artículos con la etiqueta correspondiente.

Barra lateral:  
Incluirla con la etiqueta correspondiente y con varios elementos.

Pie de pagina:  
Incluir el pie de pagina con varios contenido y con textos editados para facilitar la lectura, con su etiqueta(s) correspondientes.

Si tienes alguna duda en relacion al camino a tomar , NO procedas sin que previamente acordemos el camino. Tambien, por supuesto puedes dar sugerencias
pero NO las desarrolles hasta que lo acordemos.
