Correr PHP en Visual Studio Code

1.	Arrancamos XAMPP

2.	Apache debe estar en verde…con eso es suficiente.
	
3.	En VS Code , en la carpeta del proyecto:

a.	Seleccionamos el archivo index.php
b.	Botón derecho sobre las líneas de código del index.php y se abrirá un menú vertical:
c.	**SELECCIONAMOS: PHP SERVER: SERVER PROJECT**
* La extension PHP Server - De brapifra debe estar instalada.
	
4.	Abrira automáticamente la direccion en el navegador:
http://localhost:3000/index.php

y veremos el index.php corriendo.
