# Tema de este breve HTML - CSS : Animacion.

Las animaciones con CSS se hacen a traves del navegador y GPU. Son màs eficientes que usar JavaScript.

La idea es bajar una imagen.png de una pelota, mostrarla en el HTML, con la clase pelota.

Usando CSS  a la clase pelota le asignaremos una animation ( Animación ).

Para crear la animacion, dentro de la propiedades que asignaremos a la clase .pelota, estará la propiedad animation, cuyo
primer parámetro es el nombre de la animación que se aplicará.

La animacion como tal hay que definirla en la  etiqueta @keyframes [Nombre Animacion] { aqui las propiedades }
