# en que consiste este proyecto:
Reto: Hacer una web responsiva que explique brevemente los principios del Bushido. Los principios son: Rectitud-  (Gi en japones), Valor (Yu), Compasión (Jin), Respeto (Rei), Sinceridad (Makoto), Honor (Meiyo), Lealtad (Chugi).   

La web tiene que utilizar Flexbox y Grid Layout para adaptarse a diferentes dispositivos y tamaños de pantalla. Debe incluir al menos tres puntos de interrupción (breakpoints) con media queries. Tiene que verse perfectamente en dispositivos móviles, tabletas y computadoras de escritorio.  

Usar BEM para la designación del nombre de las clases para el  CSS. No usaremos Bootstrap.

Comentar lo principal.

Vamos a hacerlo Paso a paso, para Junior ( y/o Junior+).


# Estrategia de Desarrollo
Fase 1: Estructura Semántica (HTML5) y BEM. Definir el esqueleto y las clases.

Fase 2: Estilos Globales y Flexbox. Layout básico, tipografías y el menú/header.

Fase 3: El Grid de los Principios. El corazón de la web donde aplicaremos el diseño responsivo.

Fase 4: Media Queries y Ajustes Finales. Refinar los 3 breakpoints.
